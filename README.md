# 📌 Overview

This project presents a **Deep Learning-based Breast Cancer Classification System** using **Histopathology Images**.  
The system automatically classifies breast tissue images into:

✅ **Benign (Non-Cancerous)**  
❌ **Malignant (Cancerous)**  

The project uses both:

- 🧠 **Custom CNN Architecture**
- 🚀 **ResNet50 Transfer Learning Model**

to compare performance and improve cancer detection capability.

---

# 🎯 Project Objectives

✔ Detect breast cancer from histopathology images  
✔ Compare Custom CNN with ResNet50  
✔ Improve classification using data augmentation  
✔ Analyze performance using medical metrics  
✔ Explore AI applications in healthcare

# 🧠 Deep Learning Models

## 🔹 Custom CNN
A Convolutional Neural Network designed from scratch using:
- Conv2D Layers
- MaxPooling
- Dropout
- Dense Layers
- ReLU Activation

---

## 🔹 ResNet50 (Transfer Learning)
A pretrained deep learning architecture used for:
- Advanced feature extraction
- Better generalization
- Improved malignant detection
